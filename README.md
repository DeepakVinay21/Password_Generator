# 🔑 Password Generator (Python + Tkinter)

## 📌 Project Description
A simple **Password Generator** built using **Python and Tkinter** that allows users to create strong and secure passwords with ease.  
The application generates random passwords containing uppercase, lowercase, digits, and special characters.

## 🚀 Features
- User-friendly **GUI with Tkinter**.
- Input for **username** (optional validation).
- Choose **password length**.
- Generates **random secure password**.
- **Clear** and **Reset** options.
- Status messages for feedback.

## 🛠️ Technologies Used
- Python (Tkinter for GUI)
- Random & String modules
